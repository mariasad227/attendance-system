## Features Implemented
- Role-based authentication (Admin, Professor, Student)
- Real-time attendance marking with participation
- Justification submission with file upload
- Import/Export student lists (Progres Excel format ready)
- Responsive mobile-first design with jQuery
- Modern dashboard with charts & animations
- Full MariaDB/MySQL backend with proper constraints

## Technologies Used
- **Frontend:** HTML5, CSS3, jQuery, Chart.js
- **Backend:** PHP 8+, PDO
- **Database:** MariaDB/MySQL
- **Design:** Canva Mockup (mobile-first)
